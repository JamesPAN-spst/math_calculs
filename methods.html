反常积分敛散性：从基础到竞赛的降维打击

教练寄语： 掌握反常积分的敛散性，本质上是掌握“比较”的艺术。你要学会看透函数的“阶”（Order of Growth）。无穷远处的决战，只有增长最快或衰减最猛的函数才能掌握话语权。

第一阶：终极基石 —— 柯西收敛准则

柯西收敛准则 (Cauchy Criterion)

反常积分 $$\int_{a}^{+\infty} f(x) dx$$ 收敛的充分必要条件是：$$\forall \epsilon > 0, \exists M > a$$, 使得当 $$A_2 > A_1 > M$$ 时，都有： $$\left| \int_{A_1}^{A_2} f(x) dx \right| < \epsilon$$

例题 1.1 (反证法神兵)：证明调和积分发散

题目：证明 $$\int_{1}^{+\infty} \frac{1}{x} dx$$ 发散。

解析：利用柯西准则的否定形式。取 $$A_1 = A$$, $$A_2 = 2A$$。 $$\int_{A}^{2A} \frac{1}{x} dx = \ln(2A) - \ln A = \ln 2$$ 由于无论 $$A$$ 取多大，这段积分的值恒为 $$\ln 2$$，无法使其小于任意小的 $$\epsilon$$，故破坏了柯西准则，积分发散。

第二阶：常备武器 —— 非负函数判别法

当被积函数恒为正时，问题转化为寻找上界。我们通常使用极限比较判别法，并以 $$p$$-积分作为标尺。

例题 2.1 (多项式大乱斗)：提取最高阶

题目：判别 $$\int_{1}^{+\infty} \frac{x^2 + 2x}{x^4 + 3x^2 + 1} dx$$ 的敛散性。

解析：当 $$x \to +\infty$$ 时，分子等价于 $$x^2$$，分母等价于 $$x^4$$。 取 $$g(x) = \frac{x^2}{x^4} = \frac{1}{x^2}$$。 因为 $$\lim_{x \to +\infty} \frac{f(x)}{g(x)} = 1$$，且 $$\int_{1}^{+\infty} \frac{1}{x^2} dx$$ 收敛（$$p=2 > 1$$），故原积分收敛。
例题 2.2 (强基经典)：指数的“降维打击”

题目：判别 $$\int_{1}^{+\infty} x^{2026} e^{-0.5x} dx$$ 的敛散性。

解析：指数函数的衰减永远秒杀多项式的增长。取 $$g(x) = \frac{1}{x^2}$$。 考察极限：$$\lim_{x \to +\infty} \frac{x^{2026} e^{-0.5x}}{1/x^2} = \lim_{x \to +\infty} \frac{x^{2028}}{e^{0.5x}} = 0$$ 由极限比较判别法，原积分收敛。
例题 2.3 (瑕点陷阱)：隐藏在趋于0处的发散

题目：判别 $$\int_{0}^{1} \frac{1}{\ln(1+\sqrt{x})} dx$$ 的敛散性。

解析：注意积分区间是有限的，$$x \to 0^+$$ 是瑕点。 利用泰勒展开（等价无穷小）：当 $$x \to 0^+$$ 时，$$\ln(1+\sqrt{x}) \sim \sqrt{x} = x^{1/2}$$。 所以被积函数等价于 $$\frac{1}{x^{1/2}}$$。因为 $$p = 1/2 < 1$$，在有限区间瑕积分中，这意味着收敛！

第三阶：重型武器 —— 狄利克雷与阿贝尔

当函数符号交替，比较判别法失效时，必须祭出这两大定理处理条件收敛问题。

判定口诀
  • 狄利克雷 (Dirichlet)： 积分有界 + 单调趋近于 0。
  • 阿贝尔 (Abel)： 积分收敛 + 单调有界。
例题 3.1 (祖师爷级例题)：狄利克雷积分

题目：判别 $$\int_{1}^{+\infty} \frac{\sin x}{x} dx$$ 的敛散性。

解析：令 $$f(x) = \sin x$$, $$g(x) = \frac{1}{x}$$。 1. $$\int_{1}^{A} \sin x dx = \cos 1 - \cos A$$，其绝对值恒小于等于 2，积分有界。 2. $$g(x) = \frac{1}{x}$$ 在 $$[1, +\infty)$$ 上单调递减且趋于 0。 满足狄利克雷判别法，原积分收敛（条件收敛）。
例题 3.2 (组合拳)：阿贝尔的补刀

题目：判别 $$\int_{1}^{+\infty} \frac{\sin x}{x} \arctan x dx$$ 的敛散性。

解析：令 $$f(x) = \frac{\sin x}{x}$$, $$g(x) = \arctan x$$。 1. 由例题 3.1 已知，$$\int_{1}^{+\infty} \frac{\sin x}{x} dx$$ 收敛。 2. $$g(x) = \arctan x$$ 在 $$[1, +\infty)$$ 上单调递增，且有界（极限为 $$\pi/2$$）。 满足阿贝尔判别法，原积分收敛。

第四阶：高阶视角 —— 拆分与分部积分

例题 4.1 (CMC/考研压轴)：菲涅尔积分的降阶法

题目：证明 $$\int_{1}^{+\infty} \sin(x^2) dx$$ 收敛。

解析：被积函数连趋于0都没有，但它的振荡频率越来越快！我们用分部积分强行造出一个趋于0的项： $$\int_{1}^{A} \sin(x^2) dx = \int_{1}^{A} \frac{1}{2x} d(-\cos(x^2))$$ $$= \left. -\frac{\cos(x^2)}{2x} \right|_{1}^{A} - \int_{1}^{A} \frac{\cos(x^2)}{2x^2} dx$$ 第一项当 $$A \to +\infty$$ 时极限为 0。第二项被积函数绝对值 $$\le \frac{1}{2x^2}$$，由比较判别法绝对收敛。故原积分收敛。
例题 4.2 (大乘境界)：奇点拆分与综合应用

题目：讨论参数 $$p$$ 的取值范围，使得 $$\int_{0}^{+\infty} \frac{\sin x}{x^p} dx$$ 收敛。

解析：同时存在瑕点 $$x=0$$ 和无穷区间，必须拆分！拆为 $$\int_{0}^{1}$$ 和 $$\int_{1}^{+\infty}$$。

1. 右半段： $$\int_{1}^{+\infty} \frac{\sin x}{x^p} dx$$
由狄利克雷判别法，只要 $$\frac{1}{x^p}$$ 单调趋于 0 即可，故要求 $$p > 0$$。

2. 左半段 (瑕积分)： $$\int_{0}^{1} \frac{\sin x}{x^p} dx$$
当 $$x \to 0^+$$ 时，$$\sin x \sim x$$，所以被积函数等价于 $$\frac{x}{x^p} = \frac{1}{x^{p-1}}$$。 为了使瑕积分收敛，必须满足 $$p - 1 < 1$$，即 $$p < 2$$。

综合结论： 必须同时满足左右两段收敛，故 $$0 < p < 2$$。
